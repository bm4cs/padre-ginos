A React v18 refresher playground.

Thanks to [Complete Intro to React v9 by Brian Holt](https://react-v9.holt.courses/)

## Running

### API

A little Fastify API and SQLite DB:

```sh
git clone https://github.com/btholt/citr-v9-project.git
cd ./citr-v9-project/api
npm install
npm run dev
```

### React UI (this repo)

```sh
git clone https://github.com/bm4cs/padre-ginos.git
npm install
npm run dev
```
